# Modul: Grundlegender Schraubenmechanismus (Screw Mechanism)  
ID: mech_screw_basic  
Kategorie: Mechanik / Bewegungswandler  
Version: 1.0

## Funktion  
Umwandlung von Rotationsbewegung in lineare Bewegung oder Erzeugung großer axialer Kräfte.

## Prinzip  
Eine Schraube ist eine schiefe Ebene, die um einen Zylinder gewickelt ist.  
Durch Drehen entsteht eine axiale Verschiebung proportional zur Steigung des Gewindes.

## Inputs  
- Rotationsbewegung  
- Drehmoment  

## Outputs  
- lineare Verschiebung  
- axialer Druck oder Zugkraft  

## Typische Anwendungen  
- Schraubverbindungen (Befestigung)  
- Spindelantriebe  
- Wagenheber  
- Schraubpressen  
- CNC-Schrittschrauben (Leadscrews)  

## Abhängigkeiten

## Mechanismus  
1. Drehmoment wird an der Schraube oder Mutter angelegt  
2. Gewindesteigung bestimmt Weg pro Umdrehung  
3. Hoher Kraftgewinn möglich durch geringe Steigung  
4. Reibung wirkt wesentlich (Selbsthemmung oder gleitende Bewegung)  

## Kombinierbarkeit  
- Linearantriebe  
- Präzisionspositionierung  
- Hubmechanismen  
- Klemmsysteme  
- Übersetzungsketten (Motor → Schraubspindel → Linearschlitten)
