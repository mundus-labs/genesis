# ⚡ MODULE: BASIC ENERGY CONVERSION

## ID
energy_conversion_basic

## Kategorie
Physik / Energietechnik

## Funktion
Umwandlung einer Energieform in eine andere, nutzbare Energieform.

## Prinzip
Energie kann weder erzeugt noch vernichtet werden, sondern nur umgewandelt.  
Jede Umwandlung folgt dem Energieerhaltungssatz und ist mit Verlusten verbunden  
(z. B. Reibung, Wärmeabstrahlung, elektrische Verluste).

Typische Umwandlungsprinzipien:
- mechanisch → elektrisch (Generator)
- elektrisch → mechanisch (Motor)
- chemisch → elektrisch (Batterie)
- thermisch → mechanisch (Dampfmaschine)
- Strahlung → elektrisch (Photovoltaik)

## Inputs
- eine definierte Energieform  
(z. B. Wärme, Strahlung, chemische Energie, mechanische Bewegung)

## Outputs
- umgewandelte Energieform  
(z. B. Elektrizität, mechanische Arbeit, Wärme)

## Dependencies
Keine (Grundprinzip der Physik)

## Beschreibung
Dieses Modul beschreibt das fundamentale Prinzip der Energieumwandlung.  
Alle Energietechnologien, Maschinen, Motoren, Generatoren, Batterien,  
Heizsysteme und Photovoltaiksysteme beruhen auf der Umwandlung von  
Energieformen.

Jede reale Umwandlung besitzt einen Wirkungsgrad η < 100 %.  
Die Differenz geht als entropische Wärme oder Verluste verloren.  
Das Modul bildet die Basis für komplexere Systeme wie:

- Elektromotoren  
- Generatoren  
- Wärmekraftmaschinen  
- Brennstoffzellen  
- Turbinen  
- Photovoltaikzellen  

## Beispiele
- Dampfmaschine: Wärme → mechanische Arbeit  
- Elektromotor: elektrischer Strom → mechanische Bewegung  
- Solarzelle: Sonnenlicht → elektrischer Strom  
- Generator: mechanische Bewegung → elektrischer Strom  
- Batterie: chemische Energie → elektrischer Strom  

## Quellen
- Erster Hauptsatz der Thermodynamik  
- Energieerhaltungssatz  
- Grundlagen der Energietechnik  

## TL;DR
Energie wird nie erzeugt — sie wird nur in andere Formen umgewandelt, immer mit Verlusten.
